# Project 2: LQR Cartpole

项目中包括以下文件：

- `main.py`：主程序文件，包含仿真和可视化逻辑。
- `lqr_cartpole.py`: 实现了线性二次调节器 (LQR) 控制器，需要你完成`lqr_control`函数。
- `visualization.py`：可视化模块，用于显示小车和杆的动态。

## 运行项目

确保你已经安装了必要的依赖项，然后运行以下命令：

```bash
cd ch1_ekf_slam
pip install -r requirements.txt
python main.py --debug
```
