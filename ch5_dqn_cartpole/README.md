# Project 5: DQN Cartpole

项目中包括以下文件：

- `main.py`：主程序文件，包含仿真和可视化逻辑。
- `cartpole.py`: 定义了小车-杆系统的动力学和离散步进函数。
- `dqn.py`: 实现了深度 Q 网络 (DQN) 智能体，需要你完成相关函数。
- `visualization.py`：可视化模块，用于显示小车和杆的动态。

## 运行项目

确保你已经安装了必要的依赖项，然后运行以下命令：

```bash
cd ch5_dqn_cartpole
pip install -r requirements.txt
python main.py --epoches 500
```
